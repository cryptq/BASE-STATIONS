In Microsoft Visual Studio:

1. Open menu File\New\Project
2. Select Visual Basic Projects > Windows > Windows Form Application
3. Type in Name: VBBrowser and set project to use .NET 4 click OK
4. Open menu Project\Add Existing Item, browse to MainForm.vb, click OK
5. Open menu Project\Add Reference, go to the COM tab, select GENESYS, click Select, then OK
6. Open menu Project\Properties and change Startup Object to MainForm, click OK

Note: VBBrowser.exe and Interop.GENESYS.dll will result from the above steps, but already compiled files have been supplied for your convenience.
